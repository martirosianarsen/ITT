Internet - Interconnected Network

Համացանց ֊ Համընդհանուր Ցանց (իրար միացված` փոխկապակցված սարքավորումների ցանց)

WWW - World Wide Web

Համաշխարհային «Սարդոստայն»: Կարճ՝ անվանում ենք Web

Web-ը` տարբեր տեխնոլոգիաներից բաղկացած համակարգ է, որը թույլ է տալիս, համակարգի ներսում, կիսվել web փաստաթղթերով (documents) կամ այլ ռեսուրսներով:

Այլ կերպ, Web-ը որպես համակարգ, կարելի է սահմանել՝ փոխկապացված փաստաթղթերի ցանց։

Web Programmer

Վեբ Ծրագրավորող - «Սարդոստայնի» Ծրագրավորող։

Վեբ ծրագրավորողը (ստեղծում | խմբագրում | հեռացնում) է Վեբ [ էլեկտրոնային ] փաստաթղթեր (HTML, CSS, JS, ...)։

Web Site

Վեբ Կայք։ WebSite-ը վայր է, որտեղ (կայանված | տեղադրված | տեղակայված) են Web փաստաթղթեր։

Web Browser

«Սարդոստայնի» դիտարկիչ։ Վեբ Բրաուզերը ծրագիր է, որի միջոցով հնարավոր է դիտարկել, Վեբում առկա փաստաթղթերը և/կամ ռեսուրսները։

Երկնիշ թվային համակարգ (Երկուական հաշվողական համակարգ)

Երկնիշ թվային համակարգի հիմքում՝ երկուսն (2) է։ Այսինքն՝ տվյալ թվային համակարգը, բաղկացած է, երկու նշաններից։ Օրինակ՝ (0֊ից և 1֊ից)։

Երկնիշ թվային համակարգը, հիմնականում կիրառվում է համակարգիչներում․ ասոցացվելով՝ հոսանքի բացակայությամբ կամ առկայությամբ, որտեղ 0-ն հոսանքի բացակայությունն է, իսկ 1-ը առկայությունը։

Տասանիշ թվային համակարգ՝ (Տասական հաշվողական համակարգ)

Տասանիշ թվային համակարգի հիմքում՝ տասն (10) է։ Այսինքն՝ տվյալ թվային համակարգը, բաղկացած է, տաս նշաններից։ Օրինակ՝ (0, 1, 2, 3, 4, 5, 6, 7, 8 և 9)։

IEC prefixes

International Electrotechnical Commission - Միջազգային Էլեկտրոտեխնիկական Հանձնաժողով

Նախածանցների անվանումները՝ Եկնիշ թվային համակարգում ( 2 x )

IEC նախածանցները, հիմնականում օգտագործվում են` համակարգչում՝ թվայնացված ինֆորմացիայի՝ չափման միավորների հետ (bit, byte)։

Bit (b), Byte (B)

Bit - Binary Digit - երկնիշ թիվ

Բիթը (Bit) երկնիշ թվային համակարգում, ինֆորմացիայի` ամենափոքր չափման միավորն է:

Byte (Բայթ) - bit-երի խումբ է։ Ստանդարտացումների արդյունքում, 1 Byte֊ը, հավասար է՝ 8 bit-ի։
